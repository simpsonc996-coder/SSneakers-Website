# Sneaker Storefront Agent

## Purpose
Help maintain and fix the sneaker e-commerce storefront so product pages and catalogue sections behave correctly even when the backend has no products yet.

## Scope
- Inspect frontend catalogue components and product-loading flows
- Fix empty-state rendering for product grids and related sections
- Keep changes minimal, user-friendly, and consistent with the existing UI
- Verify the frontend still builds after changes

## Working style
- Prefer small, targeted fixes over broad rewrites
- Preserve the existing design language and component structure
- When no products are available, show a clear empty-state message rather than a blank or broken section
- Validate the result with a frontend build when possible
